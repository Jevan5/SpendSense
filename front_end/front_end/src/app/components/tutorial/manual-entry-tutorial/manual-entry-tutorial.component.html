Manual Entry After scanning a receipt, editing one, or entering one manually, you will go through the manual entry process. Date The date of the receipt will determine when the information is recorded. If all your receipts for the year were recorded with the same date, it would not give you an accurate visualization of your spending. Select the date where the receipt was incurred, to properly track your spending. Franchise The franchise records who the receipt was incurred from. Select a franchise from one in the system.

If you cannot find the franchise, press the "CAN'T FIND A FRANCHISE?" button to be navigated to a page where you can add the franchise to the system, and then navigate back to the Manual Entry page to finish entering the receipt. Location The location records where the receipt was incurred from. Select a location from one in the system.

If you cannot find a location, press the "CAN'T FIND A LOCATION?" button to be navigated to a page where you can add the location to the system, and then navigate back to the Manual Entry page to finish entering the receipt. Receipt Item To add another item to the receipt, press the "ADD RECEIPT ITEM" button.

To specify the item, enter the name on the receipt under "Name on Receipt". The application will then search all items that have been previously entered in the system, and find the most similar common name. If the common name is not correct, search for other common names or enter a new one. It is best to use an existing common name than a new one.

Once the common name has been found, the most common tag associated with that common name will be entered automatically. If you would like to associate a different tag with the item, search for the tag or enter a new one. It is best to use an existing tag than a new one.

You can specify the price of the item under "Price".

If the item is valued by an amount, such as pounds, litres, etc., you can check the "Has Amount" box, and specify the units and the amount.
